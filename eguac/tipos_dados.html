# Tipos de dados

Tipo Valores
caractere * "palavra"
curto 13
inteiro 37
longo 6666
flutuante 0.13
duplo 0.37

Note que a variável do caractere precis ter * na frente, pois está sinalizando a alocação dinâmica de memória.

main
    caractere *FR = "palavra";
    
    curto M1 = 13;
    
    inteiro M2 = 37; 
    
    longo M3 = 666;
    
    flutuante T = 0.13; 
    
    duplo TD = 0.37; 
    
    retorna 0;
fim
